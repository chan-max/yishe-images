使用指南
了解如何使用 ImageMagick 图像处理服务

操作步骤

  1. 先进行健康检查，确认 ImageMagick 已安装。
  2. 上传图片文件（支持拖拽上传）。
  3. 选择需要的操作（调整大小、裁剪、旋转等）。
  4. 设置参数并执行操作。
  5. 查看处理结果并下载。

注意事项

  • 确保 ImageMagick 已正确安装并在系统 PATH 中
  • 支持格式：JPG, PNG, GIF, BMP, WEBP, SVG, TIFF
  • 上传文件大小限制：100MB
  • 处理后的文件保存在 output 目录
